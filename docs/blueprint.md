# **App Name**: Al-Birr Society

## Core Features:

- Multilingual Support: Provides complete website content in both English and Arabic.
- Dynamic News/Updates: Content Management System (CMS) to easily update news, events, and announcements on the website.
- Secure Online Donations: Implements a secure payment gateway for online donations with options for recurring donations and multiple currencies.
- Contact Form: Interactive contact form with field validation, submission confirmation, and integration with Firestore for data storage.
- Programs Showcase: Detailed presentations of society programs with photo gallery.
- Document Summarization: Generative AI feature to automatically summarize long documents (such as the bylaws PDF) into concise overviews for quicker comprehension. LLM may act as a tool to incorporate this summarization into its response or to decide if the summarization needs to happen at all.
- Firestore Integration: Database integration for storing contact form submissions and managing dynamic content.

## Style Guidelines:

- Primary color: Deep Blue (#3B5998), evoking trust and stability, reflecting the society's established presence.
- Background color: Very light blue (#EBF2F8), provides a clean, uncluttered backdrop for content.
- Accent color: Warm Gold (#FFC107), symbolizes generosity and hope, used for key interactive elements like buttons and links.
- Headline font: 'Belleza' (sans-serif), for headlines and short chunks of text. Body font: 'Alegreya' (serif), for longer text.
- Use clear, universal icons to represent different program areas. Ensure RTL compatibility for Arabic interface.
- Responsive grid layout with clear hierarchy and consistent spacing for readability. RTL-aware design for the Arabic version.
- Subtle transitions and animations to enhance user experience, like sliding panels, fading elements and progress indicators during donation.
